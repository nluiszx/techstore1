# TechStore 🖥️

Proyecto de tienda web desarrollado con HTML, CSS y JavaScript.

## 🚀 Funcionalidades
- Catálogo de productos
- Carrito de compras
- Modal interactivo
- Diseño responsivo con Bootstrap

## 🧠 Normalización de datos

Este proyecto implementa una simulación de base de datos en JavaScript cumpliendo:

### ✔ Primera Forma Normal (1FN)
Los datos son atómicos, sin listas dentro de un mismo campo.

### ✔ Segunda Forma Normal (2FN)
Los datos dependen completamente de su clave primaria, separando productos, carrito y cliente.

### ✔ Tercera Forma Normal (3FN)
No existen dependencias transitivas, evitando redundancia de datos.
